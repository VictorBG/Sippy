Detailed Description

Author: Victor Blanco

Base implementation to test an algorithm.

A subclass must implement the getInputs() and getOutputs() methods, every position of one list must corresponds to the same position of the other list, then when {} is called, the BaseAlgorithm#decode(byte[])} and { BaseAlgorithm#encode(byte[])} method of the BaseAlgorithm} provided are tested based on the values of the input and output lists.
