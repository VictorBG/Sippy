ItemBO prop.datos.controllers.FilesController.getFile ( String  path) throws IOException

Llegeix el path i retorna un ItemBO el qual conté l'estructura del fitxer referenciat pel path. En cas de que sigui una carpeta, retorna un FolderBO que conté la estructura de la carpeta desplegada.
