Detailed Description

La clase no es usada ja que l'algorisme automatic esta "hardcodejat" al LZW per a txt per peticio del professor.

Author: Victor Blanco

Automatic algorithm chooser that chooses the algorithm based on the best algorithm for the given input.

It iterates for all of the allowedAlgorithms and encodes a small fragment of the data, which is 1KB if the file is larger than that or the whole file otherwise, then it returns the input encoded with the best algorithm based on the major percentage of encoding.
