\doxysubsection{Detailed Description}
The
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classprop_1_1algorithms_1_1lzss_1_1_k_m_p_ac8bd98eae6a339cdd35f9998b803f08d}{KMP}} }
\end{DoxyCode}
 class finds the first occurrence of a pattern string in a text string. 

This implementation uses a version of the Knuth-\/\+Morris-\/\+Pratt substring search algorithm. The version takes time proportional to {\itshape n} + {\itshape m R} in the worst case, where {\itshape n} is the length of the text string, {\itshape m} is the length of the pattern, and {\itshape R} is the alphabet size. It uses extra space proportional to {\itshape m R}. 

For additional documentation, see \href{https://algs4.cs.princeton.edu/53substring}{\texttt{ Section 5.\+3}} of {\itshape Algorithms, 4th Edition} by Robert Sedgewick and Kevin Wayne.
