\doxysubsection{Detailed Description}
Author\+: Victor Blanco 

Base implementation to test an algorithm. 

A subclass must implement the \mbox{\hyperlink{classprop_1_1algorithms_1_1base_1_1_base_algorithm_test_a3e5cd49877298ac102709db40c46b0d9}{get\+Inputs()}} and \mbox{\hyperlink{classprop_1_1algorithms_1_1base_1_1_base_algorithm_test_a692c559f79ebb855bef4fecb338674d2}{get\+Outputs()}} methods, every position of one list must corresponds to the same position of the other list, then when \{\mbox{\hyperlink{classprop_1_1algorithms_1_1base_1_1_base_algorithm_test_ad094da463e3ff06c471086fe8fa38d95}{\} is called, the  Base\+Algorithm\#decode(byte\mbox{[}$\,$\mbox{]})\} and \{ Base\+Algorithm\#encode(byte\mbox{[}$\,$\mbox{]})\} method of the  Base\+Algorithm\} provided are tested based on the values of the input and output lists. }}
