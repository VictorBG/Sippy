\doxysubsection{Detailed Description}
La clase no es usada ja que l\textquotesingle{}algorisme automatic esta \char`\"{}hardcodejat\char`\"{} al L\+ZW per a txt per peticio del professor. 

Author\+: Victor Blanco

Automatic algorithm chooser that chooses the algorithm based on the best algorithm for the given input.

It iterates for all of the \mbox{\hyperlink{classprop_1_1algorithms_1_1automatic_1_1_automatic_a02d1fef657392243edd17da27988d662}{allowed\+Algorithms}} and encodes a small fragment of the data, which is 1KB if the file is larger than that or the whole file otherwise, then it returns the input encoded with the best algorithm based on the major percentage of encoding.
